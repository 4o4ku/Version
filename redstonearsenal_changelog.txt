1.0.0:
----------------------------------------------------------------------------------------------------
Behold, the Redstone (Flux) Revolution!

GENERAL:
	-Pretty much all of this is new. It's meant to showcase how some of the new Redstone Flux (RF)
		API systems work.

ADDED:
	Blocks:
		-Added two "Fluxed" Blocks - Electrum and Crystal.
			These damage entities but infuse Redstone Flux into carried items.

	Items:
		-Full equipment set of Flux-Infused Tools.
			This includes a Pickaxe, Axe, Shovel, Sword, Sickle, Fishing Rod, and Bow.
			These tools can be empowered to have different and/or more powerful effects.
